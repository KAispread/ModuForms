# ModuForm 

ModuForm은 설문조사를 위한 웹 어플리케이션입니다. 

> **사용한 기술 스택 및 버전은 다음과 같습니다.**
- JAVA 11
- Spring Boot (2.7.3 REALEASE)
- Spring Security (5.7.3)
- Spring Data JPA (2.7.3)
    - OracleDB, H2(2.1.4) 
-  AWS (EC2)
